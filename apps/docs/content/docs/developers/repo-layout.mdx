## Точки входа в исполнение

| Что | Где | Когда вызывается |
|-----|-----|------------------|
| `npx girl-agent` | `src/cli.ts → main()` | WebUI (по умолчанию) |
| `npx girl-agent server …` | `src/server.ts → runServer()` | non-TTY automation |
| `npx girl-agent --json-events` | `src/headless.ts → runHeadlessJsonEvents()` | desktop wrapper |
| `npx girl-agent update` | `src/migrations/index.ts → runMigrations()` | apply data migrations |
| `npx girl-agent addon pack/init` | `src/cli.ts → runAddonCommand()` | dev-утилиты для аддонов |

## Что не входит в коммит

`dist/`, `webui/dist/`, `node_modules/`, `data/`, `.dev-data/`, IDE-папки. Подробности — в `.gitignore`.
