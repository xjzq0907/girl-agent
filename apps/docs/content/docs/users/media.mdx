## Что она «видит»

| Тип входящего      | В bot-режиме                    | В userbot-режиме                |
| ------------------ | ------------------------------- | ------------------------------- |
| Текст              | да                              | да                              |
| Фото               | да (через vision-модели)        | да                              |
| Стикеры            | видит как стикер с emoji        | да + видит set name             |
| Голосовое          | транскрибируется через Whisper  | транскрибируется                |
| Видео              | первый кадр + длительность      | первый кадр + длительность      |
| Кружок (видеосообщ) | транскрипт + первый кадр        | то же                           |
| Документ           | название + первые байты         | то же                           |
| GIF                | первый кадр                     | первый кадр                     |
| Геолокация         | да, координаты                  | да                              |
| Реакция            | да                              | да + кастомные                  |
| Удаление сообщения | да                              | да + увидит «удалил» в реалтайме |

## Vision (понимание фото)

Если в текущей LLM-модели есть **vision** (`gpt-4o`, `gpt-5`, `claude-sonnet-4.6`, `gemini-2.5`,
`grok-4`) — она реально **смотрит фото** и реагирует:

- Селфи → может оценить, прокомментировать, спросить где это
- Скрин → прочитает текст, отреагирует
- Мем → постарается понять и ответить в тон
- Еда → может прокомментировать что любит/не любит

Без vision-модели — увидит только «фото», без понимания. Будет отвечать абстрактно типа
«ого, красиво».

**Vision не во всех моделях.** В `gpt-4o-mini`, `claude-haiku-4.5` — есть. В `llama-3.x`,
`deepseek-chat` — нет. Если хочешь чтобы видела фото — выбери vision-модель.

## Голосовые (Whisper)

Если у тебя LLM-провайдер с поддержкой Whisper (OpenAI, Groq), или включён локальный
Whisper — голосовые **транскрибируются в текст** и она отвечает на сказанное.

В config:
```json
{
  "whisper": {
    "enabled": true,
    "model": "whisper-1",
    "baseURL": "https://api.openai.com/v1"
  }
}
```

Без Whisper — она увидит «(голосовое сообщение, 12 сек)» и ответит что-то нейтральное.

## Отправка медиа от неё

### Стикеры
В **обоих режимах** она может отправить стикер. По дефолту используется стандартный
анимированный набор Telegram + emoji-based selection через LLM.

Если хочешь свой набор — в `data/<slug>/persona/sticker-set.txt` укажи short name стикерпака
(например, `AnyaMood`), и она будет использовать его.

Принудительно отправить — `:sticker happy` (тег настроения).

### Реакции
Часто она отвечает **реакцией** вместо текста: «👍», «❤️», «😂», «🤔» — особенно если её
сообщение зацепило, но отдельный текст не нужен.

В **userbot-режиме** — может ставить **кастомные платные реакции**, если они доступны на
её аккаунте.

### Фото от неё
В дефолте она **не отправляет своих фото**. Это сознательное ограничение — генерить фото
лица через AI и выдавать за свои — это и кринж, и юридически серое.

Если ОЧЕНЬ хочется — можно подключить аддон с фотобанком (например, `addon-photos`). См.
[Addons](/docs/users/addons).

### Голосовые от неё
По дефолту — не отправляет. Можно подключить TTS-аддон (ElevenLabs, OpenAI TTS, Yandex
TTS), но он не входит в core — это extra.

## Удалённые сообщения

В **userbot-режиме** движок видит **в реалтайме**, что ты удалил сообщение. Реакция:

- Если успела прочитать → может прокомментировать («что ты там удалил-то»)
- Если не успела → не заметит
- Если ты постоянно удаляешь → копит annoyance

См. `src/engine/deletion-handler.ts` если интересно как это устроено.

## Редактирование сообщений

В userbot-режиме она **может сама редактировать** свои сообщения (как живой человек,
который опечатался / передумал):

- Если опечаталась → редактирует в течение 1-30 секунд
- Если передумала формулировку → может пересобрать в другой
- Не злоупотребляет — обычно 1-3% сообщений

В bot-режиме редактирования нет — Bot API ограничен.

## Лимиты

- **Файлы > 50 МБ** — не парсятся через Bot API. В userbot — до 2 ГБ.
- **Видео > 60 секунд** — берётся только первый кадр + длительность
- **Голосовые > 5 минут** — обрезаются для Whisper (или ошибка transcoding)

## Безопасность

<Callout type="warn">
  Все фото и голосовые **уходят к LLM-провайдеру**. Если шлёшь приватные фото — они
  пройдут через OpenAI/Anthropic/etc. Не присылай ничего, что не должно лежать на их
  серверах. Логи запросов хранятся в логах провайдера согласно их retention policy.
</Callout>
