**Профиль общения** — это пакет настроек, который определяет, **как** она пишет: длинно
или коротко, часто или редко, тёплая или холодная. Это **не персонаж** (это в `persona.md`),
а именно **манера** общения.

Профиль = 4 параметра:

- **notifications** — насколько внимательно следит за уведомлениями (`muted` / `normal` / `priority`)
- **messageStyle** — стиль сообщений (`one-liners` / `balanced` / `bursty` / `longform`)
- **initiative** — как часто пишет первой (`low` / `medium` / `high`)
- **lifeSharing** — насколько охотно делится тем что происходит у неё (`low` / `medium` / `high`)

## 5 готовых пресетов

### Нормальная (`normal`) — дефолт

> золотая середина — отвечает нормально, не липнет, иногда сама пишет

| Параметр       | Значение  |
| -------------- | --------- |
| notifications  | normal    |
| messageStyle   | balanced  |
| initiative     | medium    |
| lifeSharing    | medium    |

- Отвечает в среднем за минуту-полчаса
- Сообщения по 1-3 предложения, иногда длиннее если есть что рассказать
- Может написать первой раз в день-два, без надоедания
- Делится бытовым по ситуации — не вываливает поток жизни, но и не молчит

**Когда выбирать:** если не знаешь что выбрать. Это нейтральный дефолт, который не палит ни
в одну крайность.

---

### Милая (`cute`)

> тёплая и заботливая, часто отвечает, пишет первой, делится моментами

| Параметр       | Значение  |
| -------------- | --------- |
| notifications  | priority  |
| messageStyle   | balanced  |
| initiative     | high      |
| lifeSharing    | high      |

- Быстро видит сообщения, отвечает охотно
- Часто инициатива от неё: «как ты», «что делаешь», «скучаю» и т.п.
- Делится тем что у неё на душе
- Не липнет в смысле «спам пузырями», но эмоционально близка

**Когда выбирать:** хочешь, чтобы тебе уделяли внимание. Подходит для стадий 4-7
(`convinced` → `dating-stable`).

<Callout type="info">
  «Милая» — это не «приторная». Она всё ещё может ревновать, обижаться, иметь свои дела. Это
  не sweet-ассистент.
</Callout>

---

### Альтушка (`alt`)

> холодная, сухая, короткие ответы, почти не пишет первой, личным не делится

| Параметр       | Значение     |
| -------------- | ------------ |
| notifications  | normal       |
| messageStyle   | one-liners   |
| initiative     | low          |
| lifeSharing    | low          |

- Ответы по 1-3 слова, односложные («ну», «да», «норм», «хз»)
- Очень редко пишет первой
- Не делится своей жизнью, на «что у тебя» — «всё.»
- Чаще игнорит, может «прочитать и не ответить»

**Когда выбирать:** хочешь стереотип «холодной альтушки» / «недоступной». Имей в виду — она
сложнее «открывается», нужно будет «грести» к ней.

---

### Залипала (`clingy`)

> очень липучая, спамит пузырями, всегда онлайн, пишет первой постоянно

| Параметр       | Значение  |
| -------------- | --------- |
| notifications  | priority  |
| messageStyle   | bursty    |
| initiative     | high      |
| lifeSharing    | high      |

- Длинная мысль разбивается на 3-5 пузырей подряд
- Постоянно пишет — если ты молчишь 10 минут, может прислать «эй»
- Очень эмоциональна, ревнует, обижается быстро
- Может звонить и спамить если игнорить долго

**Когда выбирать:** хочешь экстрим в сторону «навязчивости». Подходит для стадий 5-7. На
ранних — будет нереалистично.

<Callout type="warn">
  С Залипалой нужно учиться **отвечать**, иначе быстро прокачивается **annoyance** и она
  обижается / молчит сутками. См. [Конфликт →](/docs/users/conflict).
</Callout>

---

### Болтушка (`chatty`)

> любит рассказывать истории, пишет длинные тексты, часто делится бытовым

| Параметр       | Значение  |
| -------------- | --------- |
| notifications  | priority  |
| messageStyle   | longform  |
| initiative     | medium    |
| lifeSharing    | high      |

- Длинные сообщения с историей: «короче, сегодня в метро было...»
- Часто пересказывает что случилось у неё — пары, работа, подруги
- Не липнет, но **расскажет** всё что происходило
- Может оставить тебя «слушать» долго

**Когда выбирать:** если хочешь много контента, истории из её «жизни», ощущение что она
**проживает день** и тебе это рассказывает.

---

## Что значит каждый параметр

### notifications

Насколько быстро она замечает твоё сообщение в принципе.

| Значение  | Эффект                                                          |
| --------- | --------------------------------------------------------------- |
| muted     | может вообще не заметить часами, телефон где-то лежит           |
| normal    | в среднем замечает за 1-15 минут, как обычный человек           |
| priority  | у неё уведомления о тебе включены, видит почти сразу            |

### messageStyle

Длина и форма её сообщений.

| Значение     | Эффект                                              |
| ------------ | --------------------------------------------------- |
| one-liners   | одно-два слова, «хз», «ну»                          |
| balanced     | 1-3 предложения по ситуации                         |
| bursty       | длинная мысль разбивается на 3-5 пузырей подряд     |
| longform     | большие сообщения с историями                       |

### initiative

Как часто пишет первой.

| Значение  | Эффект                                                       |
| --------- | ------------------------------------------------------------ |
| low       | редко, только в особых случаях                               |
| medium    | раз в день-два может что-то спросить или рассказать          |
| high      | сама поддерживает темп: «как ты», «что делаешь», скидывает мемы |

### lifeSharing

Насколько охотно делится своей жизнью.

| Значение  | Эффект                                                      |
| --------- | ----------------------------------------------------------- |
| low       | «всё норм» и без подробностей                               |
| medium    | расскажет о важном, на бытовое не разводится                |
| high      | вываливает день, эмоции, чай с подругой и т.п.              |

## Можно ли кастомизировать?

Да. В WebUI → **Config** → раздел **Communication** — можно менять каждый параметр
отдельно, без привязки к пресету. Это сохранится как кастомный профиль.

Также через CLI/env:

```bash
GIRL_AGENT_COMM_PRESET=alt girl-agent server --headless
```

## Как это влияет на промпт

Профиль превращается в инструкции в системном промпте — `communication.md` файле в папке
профиля. Можно открыть его в **Memory** и отредактировать вручную для более тонкого
поведения. См. [Memory](/docs/users/memory).
