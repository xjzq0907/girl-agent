## Что важно понять с самого начала

### 1. Это не чат-бот «всё-знаю»

`girl-agent` — это **персонаж**, который живёт в Telegram. Она:
- может **игнорить**, если ты её бесишь
- может **спать** ночью и не отвечать
- может **обижаться** и молчать часами
- может **читать сообщение и не отвечать** (но галочки прочитанного появятся)
- может **писать первой**, если давно не общались

Это не баг. Так задумано. Не пытайся выжать из неё «нормальное ассистент-поведение» —
не получится.

### 2. Есть два режима в Telegram

- **Bot-режим** (`@BotFather` создаёт токен) — проще, безопаснее, но видно что это бот.
- **Userbot-режим** (твой второй Telegram-аккаунт через API ID/Hash) — выглядит как живой
  человек, может ставить реакции, читать, удалять, редактировать. **Используй отдельный
  тестовый аккаунт — Telegram может забанить за подозрительную активность.**

Подробнее: [Bot vs Userbot →](/docs/users/telegram-modes)

### 3. Тебе нужен ключ от LLM

Сам по себе `girl-agent` не думает — он обращается к ИИ (OpenAI, Anthropic, ClaudeHub и др.).
Тебе нужен API-ключ от одного из них. Рекомендуем **ClaudeHub** — работает в РФ через СБП и крипту.

Подробнее: [LLM-провайдеры →](/docs/users/llm-providers)

### 4. Все настройки — в WebUI

После установки запускаешь `girl-agent`, открывается **WebUI** на `http://localhost:3000`.
Там визард первой настройки, профили, лог, настройки, маркетплейс аддонов — всё через
браузер. Терминал нужен только для запуска.

Подробнее: [WebUI →](/docs/users/webui)
