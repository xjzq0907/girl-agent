`girl-agent` сам не «думает» — он отправляет запросы к языковой модели (LLM) у внешнего
провайдера. Это твой выбор: рекомендуем **ClaudeHub** для России, **Groq** или **DeepSeek**
для скорости/цены, **Ollama** для локального запуска.

Все провайдеры с готовыми пресетами доступны в выпадушке мастера. Полный список ниже.

## Рекомендуемые провайдеры

### ClaudeHub
- **Протокол:** Anthropic (Claude) + OpenAI-compatible
- **Адрес:** `https://api.claudehub.fun`
- **Модели:** `claude-sonnet-4.6` (по умолчанию), `claude-opus-4.7/4.6/4.5`, `claude-haiku-4.5`, `gpt-5.5`, `gpt-5.4`, а также **`gemini-3.1-pro-preview`**
- **Цены:** **до 150× дешевле** прямых OpenAI/Anthropic — это главная фишка, не просто "альтернативный шлюз".
- **Оплата:** **СБП, крипта, российские карты**
- **Регистрация:** [claudehub.fun](https://claudehub.fun)

Почему рекомендуем — единственный нормальный способ из РФ платить за Claude / GPT / Gemini
без VPN и зарубежных карт, плюс экономия на API в десятки раз при том же качестве и
близкой скорости.

### Groq
- **Протокол:** OpenAI-compatible
- **Адрес:** `https://api.groq.com/openai/v1`
- **Модели:** `llama-3.3-70b-versatile`, `llama-3.1-8b-instant`, `mixtral-8x7b-32768` и др.
- **Цены:** есть бесплатный тиер, дальше — копейки
- **Регистрация:** [console.groq.com](https://console.groq.com)

Самый быстрый — ответы в районе 500 токенов/сек. Бесплатно для тестов. Качество чуть ниже
Claude/GPT, но для тг-стиля норм.

### DeepSeek
- **Протокол:** OpenAI-compatible
- **Адрес:** `https://api.deepseek.com`
- **Модели:** `deepseek-chat`, `deepseek-reasoner`
- **Цены:** в разы дешевле OpenAI (примерно $0.14/M токенов на input)
- **Регистрация:** [platform.deepseek.com](https://platform.deepseek.com)

Дешёво и **очень хорошо для русского языка**. Reasoner — мощный режим с цепочками
рассуждения, для нашего юзкейса не обязателен.

## Прямые гиганты

### OpenAI
- **Протокол:** OpenAI
- **Адрес:** дефолтный (api.openai.com)
- **Модели:** `gpt-4o`, `gpt-4o-mini`, `gpt-4.1`, `gpt-5`, `o1`, `o3-mini` и т.д.
- **Оплата:** только зарубежная карта
- **Регистрация:** [platform.openai.com](https://platform.openai.com)

Из РФ платить нельзя напрямую — нужен VPN и зарубежная карта.

### Anthropic
- **Протокол:** Anthropic
- **Адрес:** дефолтный
- **Модели:** `claude-3.7-sonnet`, `claude-opus-4`, `claude-sonnet-4.5` и т.д.
- **Регистрация:** [console.anthropic.com](https://console.anthropic.com)

Аналогично OpenAI — оплата зарубежной картой.

### Google Gemini
- **Протокол:** OpenAI-compatible (через `/v1beta`)
- **Адрес:** `https://generativelanguage.googleapis.com/v1beta/openai`
- **Модели:** `gemini-2.5-flash`, `gemini-2.5-pro`, `gemini-2.0-flash`
- **Цены:** есть бесплатный тиер с лимитами
- **Регистрация:** [aistudio.google.com](https://aistudio.google.com/apikey)

Бесплатно с лимитами 1500 запросов/день на Flash — хороший вариант для теста.

### xAI (Grok)
- **Протокол:** OpenAI-compatible
- **Адрес:** `https://api.x.ai/v1`
- **Модели:** `grok-4`, `grok-3`, `grok-3-mini`
- **Регистрация:** [console.x.ai](https://console.x.ai)

## Дешёвые / специальные

### Mistral
- **Протокол:** OpenAI-compatible
- **Адрес:** `https://api.mistral.ai/v1`
- **Модели:** `mistral-large-latest`, `mistral-medium-latest`, `codestral-latest`
- **Регистрация:** [console.mistral.ai](https://console.mistral.ai)

### Together AI
- **Протокол:** OpenAI-compatible
- **Адрес:** `https://api.together.xyz/v1`
- **Модели:** масса опен-сорс моделей (Llama, Qwen, Mixtral)
- **Регистрация:** [together.ai](https://api.together.ai)

### Fireworks
- **Протокол:** OpenAI-compatible
- **Адрес:** `https://api.fireworks.ai/inference/v1`
- **Модели:** Llama 3.x, Qwen, DeepSeek-V3 и др.
- **Регистрация:** [fireworks.ai](https://fireworks.ai)

### Perplexity
- **Протокол:** OpenAI-compatible
- **Адрес:** `https://api.perplexity.ai`
- **Модели:** `sonar`, `sonar-pro`, `sonar-reasoning`
- **Особенность:** ответы со ссылками на источники (не оптимально для девушки в тг)

### Cerebras
- **Протокол:** OpenAI-compatible
- **Адрес:** `https://api.cerebras.ai/v1`
- **Модели:** `llama-3.3-70b`, `llama-4-scout`
- **Особенность:** ультрабыстрая, но небольшое окно контекста

## Локально / бесплатно

### Ollama
- **Протокол:** OpenAI-compatible
- **Адрес:** `http://localhost:11434/v1`
- **Модели:** что скачал (`llama3.1`, `qwen2.5`, `mistral`, `gemma2` и т.д.)
- **Цена:** бесплатно, нужна твоя видеокарта (16+ ГБ VRAM для 70B моделей)

Установка:
```bash
curl -fsSL https://ollama.com/install.sh | sh
ollama pull llama3.1:8b   # лёгкая модель
ollama pull qwen2.5:14b   # рекомендуем для русского
```

В girl-agent выбери провайдер **Ollama**, ключ можно оставить пустым.

### LM Studio
- Аналог Ollama, но с GUI
- Запусти **OpenAI-compatible server** в LM Studio → получишь `http://localhost:1234/v1`
- В girl-agent выбери **Custom OpenAI**, baseURL = `http://localhost:1234/v1`

## Custom (свой baseURL)

Если твой провайдер не в списке, но совместим с OpenAI или Anthropic API — выбери
**Custom OpenAI** или **Custom Anthropic**, введи свой `baseURL`, `model`, `apiKey`.
Работает с любым GPT-совместимым прокси/самохостом.

## Поменять модель уже после старта

- **WebUI:** профиль → **LLM** → выбрать новую модель → **Save**
- **CLI:**
  ```bash
  girl-agent --set-model --profile=anya --api-preset=claudehub --model=claude-opus-4.6 --api-key=...
  ```
- **Через config:** в `data/<slug>/config.json` поправить `llm.model` и перезапустить

## Ошибки

| Ошибка                              | Что значит                                           |
| ----------------------------------- | ---------------------------------------------------- |
| `401 Unauthorized`                  | Ключ невалидный, истёк или не тот провайдер          |
| `429 Too Many Requests`             | Превысил rate limit — подожди или смени тариф        |
| `Insufficient quota`                | Баланс закончился                                    |
| `Connection refused`                | Ollama не запущен / неверный baseURL                 |
| `Model not found`                   | Модель не доступна на этом провайдере / опечатка     |

Полный гайд: [Troubleshooting → LLM](/docs/users/troubleshooting#llm-не-отвечает)
